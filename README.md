# Dating Chat Coach：不会聊天，也能把话说对

> 你不是没有话说。你只是经常在最关键的一句话上，说成了“建议”、 “审问”或者“解释自己”。

Dating Chat Coach 是一个面向约会和相亲场景的开源 AI Skill：

- 聊天中，结合你的语言习惯和人设，生成自然、能直接发送的回复
- 根据双方的聊天上下文，判断兴趣信号、关系阶段和下一步动作
- 面对冷场、邀约、误会和拒绝时，给出有分寸的表达

它不替你表演一个完美的人，也不教你使用套路。它的目标很简单：让你更像自己，但表达得更清楚。

## 先看一个真实使用方式

把聊天截图或文字直接发给 AI：

```text
用 $dating-chat-coach 帮我回这句话。

我的语言习惯：说话比较直接，不喜欢太油腻，平时消息不长。
我的目标：先自然地聊下去，不急着邀约。

对方说：明天台风，宅家休息，做饭看看综艺。你呢？
```

它会先判断对方有没有给出话题切口，再给你一条可以直接发送的回复，并说明下一步观察什么。

输出会尽量保持这种结构：

```text
判断：对方在正常接话，也主动问了你。先回答自己，再接住她的生活安排。

建议发送：我今天也差不多，准备把节奏放慢一点。下雨天适合做点简单的饭，再看个不用太费脑子的东西。

避免：不要连续追问，也不要只回复“你准备做什么饭”，否则聊天会变成采访。
```

## 这个项目解决什么问题

很多人不是不想认真认识一个人，而是不知道怎么把真实想法说出来：

- 对方说“今天被老板骂了”，你下意识回“那你怼回去”，结果把安慰说成了评判
- 聊天一直停在“哈哈”“你吃了吗”，想推进却不知道从哪里开始
- 见完面觉得“哪里不对”，但分不清是红旗、节奏不合，还是自己过度解读
- 想建立自己的聊天风格，却每次都只能临时问一句“这怎么回”

Dating Chat Coach 把这些问题拆成可操作的步骤：上下文、语言习惯、关系阶段、对方信号、回复目标。

## 核心能力

### 1. 用你的口吻，而不是 AI 腔

你可以提供自己的语言习惯、常用表达、雷区、消息长度和希望呈现的语气。助手会据此生成回复，而不是强行把所有人写成同一种“高情商模板”。

### 2. 先判断互动，再决定怎么回

它会区分刚认识、破冰、升温、邀约、见面后、降温和修复，不把每一句话都解释成“对方喜欢你”，也不因为一条短消息就直接判定对方没兴趣。

### 3. 一条主回复，少量备选

默认给一条推荐回复，必要时再给不同语气的备选：稳重、活泼、推进或修复。重点是帮你做决定，不是制造一堆更难选择的话术。

### 4. 把边界放在套路前面

不提供胁迫、骚扰、跟踪、绕过拉黑、制造嫉妒、贬低式搭讪或身份欺骗的话术。沟通辅助应该帮助双方更清楚，而不是把操控包装得更漂亮。

## 安装聊天辅助 Skill

如果你使用支持 Codex Skill 的环境，可以把 `skills/dating-chat-coach` 放到本地 Skills 目录：

```bash
mkdir -p ~/.codex/skills
cp -R skills/dating-chat-coach ~/.codex/skills/
```

然后在对话中使用：

```text
$dating-chat-coach
```

也可以不安装，直接把 `skills/dating-chat-coach/SKILL.md` 的内容提供给你的 AI 编程助手。

## 建议先填写自己的沟通档案

模板位于 [`skills/dating-chat-coach/references/context_template.md`](skills/dating-chat-coach/references/context_template.md)。建议先补充：

- 你通常怎么说话
- 你喜欢和不喜欢哪些表达
- 你希望显得稳重、温和、直接还是活泼
- 你当前的交友目标和关系边界
- 对方的基本资料和当前关系阶段

信息越真实，生成的回复越像你。没有资料也可以直接使用，助手会只根据当前聊天做保守判断。

## 隐私提醒

聊天截图、个人资料和约会记录都可能包含敏感信息。公开分享前请删除姓名、手机号、头像、公司、住址和其他可识别信息。这个项目的通用 Skill 不读取任何固定的个人档案，也不会替你编造身份、经历或情感状态。

## 参与共建

欢迎提交 Issue 或 Pull Request：

- 更好的中文聊天场景和回复结构
- 不油腻、不操控的沟通边界
- 不同地区、平台和关系阶段的使用案例
- 对隐私、误判和安全问题的改进建议

这个项目不承诺替你找到对象，也不替你做最终决定。它只帮你把一次聊天、一次见面和一个模糊的感觉，变成更清楚的信息。

如果它帮你少说错一句话，或者让你更早看清一个不合适的人，欢迎点个 Star，让更多不会聊天但愿意认真认识别人的人看到它。
