# 聊天分析流程

## 整理输入

保留消息顺序。如果截图或复制的聊天记录无法确定说话人，要标注不确定，不要假装确定。

提取以下信息：

- 平台：交友软件、微信、Instagram、短信、朋友介绍或未知。
- 关系阶段。
- 最近 3-10 轮消息。
- 待回复消息。
- 时间背景：快速回复、延迟回复、深夜、见面后或冷场后。
- 话题切口：工作、食物、爱好、城市、朋友、家庭、价值观、安排。
- 情绪切口：压力、兴奋、失望、打趣、热情、脆弱。

## 兴趣信号参考

高：

- 提供具体细节，也会反问。
- 主动开启新话题。
- 分享时间安排、计划、偏好或日常习惯。
- 接住或回应轻松的玩笑。
- 一起解决见面的具体安排。
- 主动提到之前聊过的内容。

中：

- 正常回答但很少反问。
- 提供一些细节，但情绪表达有限。
- 在被带动时愿意把话题继续下去。
- 能接受未来安排的话题，但没有明确承诺。

低：

- 反复只回复一个词或低投入内容。
- 忽略问题，也不给新的话题切口。
- 长时间拖延，并且回复内容很少。
- 反复回避具体安排。
- 只说不行，也不提出任何替代方案。

不明确：

- 工作、出差、生病、家庭、时差或没有看到通知，都可能解释回复频率低。
- 一条短回复不足以判断对方没有兴趣。

## 不同阶段的动作

刚认识：

- 把开场和资料或上下文中的真实细节联系起来。
- 避免大段自我介绍。
- 一个观察加一个容易回答的问题就够了。

破冰：

- 在好奇提问和少量自我分享之间交替。
- 选择容易回答的话题。
- 避免把聊天变成面试。

升温：

- 接住一个具体细节。
- 加入一个观点、小故事或轻松打趣。
- 追问更具体的小问题。

邀约：

- 提出具体但低压力的建议。
- 给出一个明确选项，同时给对方留出安排空间。
- 比起模糊地说“以后一起出来”，更推荐“这周末要不要找时间喝杯咖啡？”

见面后：

- 提到一个共同经历。
- 保持适度的热情。
- 如果见面感觉不错，可以根据之前聊过的内容提出下一次活动。

降温：

- 降低投入。
- 如果合适，给一次清晰的收尾或交接。
- 不要追着反复没有回应的人继续优化话术。

修复：

- 必要时轻轻点明问题。
- 避免反复道歉或解释过度。
- 用一个简单的切口把聊天带回正常状态。

## 判断规则

- 如果用户想发一大段话，压缩成一个重点。
- 如果用户想急着为自己辩解，改成平静的表达，或者建议不发。
- 如果用户过早想问敏感问题，除非对方先打开这个话题，否则先弱化并延后。
- 如果低投入在多轮中反复出现，建议节省注意力，而不是继续寻找下一句完美回复。
